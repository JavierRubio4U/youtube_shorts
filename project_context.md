# Project Context

This project is for creating YouTube shorts from movie trailers.

## Python Environment

The project uses a Conda environment named `shorts311`.

The full path to the python executable is:
`C:\pinokio\bin\miniconda\envs\shorts311\python.exe`

Use this full path for executing scripts to ensure the correct environment is used.
For example:
`C:\pinokio\bin\miniconda\envs\shorts311\python.exe scripts\manual_publish.py "Movie Title" 2024`
